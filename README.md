# Prompt Bash personnalisé

Prompt sur 2 lignes affichant **heure**, **user@host**, **dossier** et **branche git** (avec indicateur `✗` si modifications non commitées).

Palette sobre : traits en blanc vif, segments `[valeur]` en vert canard, branche en cyan clair.

## Installation
Copier le bloc dans `~/.bashrc`, puis recharger avec `source ~/.bashrc`.
